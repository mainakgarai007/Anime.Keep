# Anime.Keep

ANIME.KEEP is a full-stack anime ecosystem starter built with Node.js + Express + vanilla frontend and dynamic Jikan API integration.

## Run

```bash
npm install
npm start
```

Open: `http://localhost:3000`

## Scripts

- `npm run lint` - syntax check
- `npm run build` - build placeholder for static architecture
- `npm test` - API smoke tests

## Features implemented

- MG STYLE AMOLED neon UI shell
- Auth (register/login/guest/remember)
- Home discovery sections from Jikan
- Anime search with filters and live suggestions
- Anime details with characters and recommendations
- Schedule (today/tomorrow/seasonal)
- Tracker CRUD
- Watchlists CRUD + share
- Community posts/comments/likes/spoilers
- Profile stats dashboard
- Notification APIs
- Settings + import/export + offline local sync
- Admin dashboard (role-protected)
